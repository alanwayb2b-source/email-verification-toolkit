# Contributing to Email Verification Toolkit

Thank you for your interest in contributing to the Email Verification Toolkit!

Contributions of all kinds are welcome, including bug fixes, new features, documentation improvements, and suggestions.

## How to Contribute

1. Fork this repository.
2. Create a new branch for your changes.
3. Make your changes.
4. Test your changes locally.
5. Commit your changes with a clear commit message.
6. Push your branch to your fork.
7. Open a Pull Request.
